Assets

Source assets for the website live here and are copied to /assets/ in the built site.

Recommended structure:

  • images/profile/ - profile photos, avatar/brand images
  • images/projects/ - project thumbnails, mockups, case-study visuals
  • images/backgrounds/ - decorative background graphics or textures
  • files/ - source downloads that should ship with the site

The generated resume PDF is still created at build time and written to docs/assets/.
